# openRigConsole

Desktop ham radio application for macOS, Linux, and Windows. Provides rig control, QSO logging, DX cluster spots, and bandmap in a single window.

## Requirements

- [Flutter](https://flutter.dev) SDK

## Building

```bash
# macOS
flutter build macos

# Linux
flutter build linux

# Windows
flutter build windows
```

## Running (development)

```bash
flutter run -d macos
```
